Gauge Types

                        Gauge directive lets you create three distinct types of gauges:
                        
    
                            
  • full (default) - a circular gauge
    • 
                            
  • semi - a semi-circular gauge
    • 
                            
  • arch - arc like gauge less than a full circle.
    • 
                        

                    

                
                
                    

                        

                    

                
            
        

        

            

                
                

                
                

                
                
            

        

    

    

        

            
                
                    

                        
Gauge Styles

                        Gauge directive provides two cap styles for Gauge lines:
                        
    
                            
  •  butt (default) - gives a flat edge to gauge line
    • 
                            
  • round - gives a rounded edge to gauge line
    •

Gauge's Scale

                        

                            The range of Gauge's scale can be specified using "min" and "max" attribute. The "value"
                            attribute specifies the
                            current value that lies between the given range is rendered on the gauge's scale.
                        

                    

                
                
                    

                        

                    

                
            
        

        

            

                
                
            

        

    

    

        

            
                
                    

                        
Dynamic Gauge Value

                        

                            The gauge value can be specified directly or bound to a scope variable defined in the

Gauge Themes

                        Themes for gauge widget can be set manually using foreground color and background color
                        attributes. In upcoming releases,
                        it will contain set of named themes which can be applied based on choice.
                        
    
                            
  •  foregroundColor - rgba(0, 150, 136, 1) (default)
    • 
                            
  • backgroundColor - rgba(0, 0, 0, 0.1) (default)
    • 
                        

                        
It can accept any valid CSS color value.
